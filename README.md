# Baza kart graficznych w C

Prosty program konsolowy napisany w języku C do zarządzania binarną bazą danych kart graficznych. Aplikacja pozwala tworzyć i otwierać pliki `.dat`, a następnie przeglądać zapisane rekordy, dodawać nowe wpisy, edytować istniejące dane, usuwać rekordy oraz sortować bazę według wybranych parametrów karty.

## Pliki
- `projekt.c` – główny program do obsługi bazy.
- `zapis.c` – program zapisujący przykładowe rekordy do `baza1.dat`.
